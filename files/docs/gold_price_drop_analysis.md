# 黄金价格下跌原因简报（近30天）

**时间范围：** 近30天（重点：2026-03-03 当日回撤）

## 核心结论（简版）
近期金价虽整体偏强，但在3月初出现显著回撤，主要由**美元走强**、**美债收益率/真实利率上行**、**美联储降息预期后移**、**风险偏好与流动性转向**等因素叠加导致。ETF与长期结构性买盘仍提供底部支撑，但短期宏观冲击占主导。

## 关键驱动因素（含来源）
1. **美元走强压制金价**：美元指数走高使美元计价的黄金对非美买家更贵，抑制需求。  
   - 来源：TradingEconomics 金价与宏观数据 https://tradingeconomics.com/commodity/gold  
   - 来源：USAGOLD 日度金价/美元联动 https://www.usagold.com/daily-gold-price-history  

2. **美债收益率/真实利率上行**：收益率上升抬高持有无息资产的机会成本。  
   - 来源：USAGOLD 日度金价/收益率讨论 https://www.usagold.com/daily-gold-price-history  
   - 来源：LiveMint 对收益率与金价的解释 https://www.livemint.com/market/commodities/gold-silver-price-today-live-march-3-2026-us-iran-war-impact-gold-2-lakh-prediction-silver-100-dollars-mcx-opening-11772502910090.html

3. **美联储降息预期后移（偏鹰预期）**：通胀风险上升/能源冲击使市场推迟降息预期，利率“更高更久”。  
   - 来源：LiveMint 相关报道 https://www.livemint.com/market/commodities/gold-silver-price-today-live-march-3-2026-us-iran-war-impact-gold-2-lakh-prediction-silver-100-dollars-mcx-opening-11772502910090.html  
   - 来源：TradingEconomics（宏观预期与金价） https://tradingeconomics.com/commodity/gold

4. **风险情绪与流动性偏好**：地缘冲突引发“美元优先”的避险偏好，部分资金转向现金而非黄金。  
   - 来源：Mining.com 报道（冲突与美元/金价反向） https://www.mining.com/gold-price-drops-6-on-war-induced-inflation-fears

5. **ETF/仓位因素**：长期ETF流入提供底部支撑，但短期回撤更多来自宏观定价变化；未见明确单一ETF大规模流出作为主导因素。  
   - 来源：Yahoo Finance（ETF/投行观点） https://sg.finance.yahoo.com/news/goldman-sachs-revamps-gold-price-174700577.html

## 结论与判断
- **短期下跌主因**：美元走强 + 收益率上行 + 降息预期后移。  
- **中期支撑因素**：央行购金、ETF长期资金、地缘不确定性。  
- **风险提示**：若美元与真实利率继续上行，金价仍有阶段性调整压力；若风险事件再升级或降息预期重新抬升，金价或恢复上行。
