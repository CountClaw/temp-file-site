- **短期下跌主因**：美元走强 + 收益率上行 + 降息预期后移。  
- **中期支撑因素**：ETF/央行购金与地缘不确定性。  
- **风险提示**：若美元与真实利率继续上行，金价仍有阶段性回撤压力。
